For mobile carousel cards (BrandCardHome in FeaturedBrandPartners), user prefers 88% card width over the original 75%.

**Why:** 88% gives a ~322px card on 375px screens, making product images ~145px (vs 108px at 75%). Feels spacious without losing the carousel peek affordance (~37px visible).

**How to apply:** When sizing mobile carousel cards, default to 85–90% width. Don't go below 80% (too narrow) or above 90% (loses scroll signal).
